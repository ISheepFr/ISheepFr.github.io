Colorful Navigation on Google Maps

Project written in C++ with OpenCV



Description

This project enables users to navigate Google Maps using hand movements and two differently colored pens. The red pen controls mouse movement, while the green pen simulates a left mouse click. The color calibration process improves accuracy.

  • Color Calibration: The user performs two calibration clicks to associate one color for clicking and another for mouse movement.

  • Mouse Control: The user moves the mouse using the second color, and a left-click is executed when both colors are detected simultaneously.
