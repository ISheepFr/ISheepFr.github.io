Interactive image viewer

Project written in C++ with OpenCV

View on GitHub


Description

The project aims to read and display all the images in a folder, and then control this "virtual image library" using a pen and color detection.

The images present in the folder are located at the bottom of the interface, and the current image is surrounded in red. The two green and red rectangles allow you to move to the next / previous image when the pen collides with the rectangles.

A first click on the pen or any object is necessary to calibrate its color.
