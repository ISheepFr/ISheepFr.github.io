Bluetooth keyboard

Project wrote in C++, with arduino

View on GitHub

Description

A simple 9 keys bluetooth keyboard with an oled screen and a button for swapping key bindings

The reason why I created this keyboard was because of one problematic i had. I'm playing various videogame with differents key bindings for each and I wanted to be able to quickly change preset. Also, i wanted to have a wireless portable keyboard.

Parts used

1 x ESP32 NodeMCU 1 x I2C OLED screen 12 x keyboard switches 1 x push button 1 x ARGB LED strip WS2811
The brain of the project, with the following roles :

  • Managing the bluetooth connection
  • Detection of the press/release of a key
  • Detection of the profile change's button
  • Managing the display on the screen
  • Managing the light effect of the strip led

    • The OLED screen display if the keyboard is currently connected by bluetooth. If he is connected, the current profile name is shown with all the current bindings associated to this profile.
    Like a mechanical keyboard, the switches are used to input. Depending on the active profile, the key bindings might change.
    This button is used to switch between all the different profile (currently 3 : Lol, Valorant, CS:GO).
    This led strip is only for the decoration. It is easily programmable for custom effect. The strip is currently making effect when the switches are pressed.
