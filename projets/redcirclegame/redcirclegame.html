Balloon Pop Simulation

Project written in C++ with OpenCV

View on GitHub


Description

This project is a balloon-popping game where the player uses a red pen to pop balloons (circles) on the screen. The objective is to increase the score by popping balloons before the screen fills up. The game involves:

  • Pen-based control: Using a red pen, the player interacts with the game, popping balloons that collide with the pen as detected by the camera.

  • Gameplay: The balloons appear randomly on the screen, and the player must pop them to score points. The difficulty level increases the speed and number of balloons.

  • Ending Condition: The game ends when the player press ESC.






Pen Detection

When the game starts, the camera feed is displayed. The player uses a red pen to pop balloons by moving it in front of the camera. The game detects the red color from the pen to determine collisions with the balloons.
