Voronoi generator

Project wrote in python


Description

The goal of this project was to implement a first bruteforce algorithm of voronoi's diagrams and be able to apply this algorithm on input image.

How it works :

  • Random generation of big amount of points (voronoi seeds)
  • Computation of Voronoi's diagram with selected seeds
  • Associate the average of the values ​​with the pixels of each region

How to use it?

Positional parameters

  • nbseeds : The number of seeds to create Required
  • --dist NUMBER (between 0,2) : The method for distance computation Optional
    0 = euclidian dist (default)
    1 = Manhattan dist
    2 = Chebyshev dist
  • --img IMG_PATH : Path of the image to aproximate with Voroni's diagram Optional

Output

If an image path parameters is filled, the output will be Voronoi's diagram on the image. If no image path is filled, the computation is done on a black background, giving us a similar output image as the Description section.
