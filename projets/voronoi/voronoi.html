Voronoi generator

Project wrote in python

Description

The goal of this project was to implement a first bruteforce algorithm of voronoi's diagrams and be able to apply this algorithm on input image.

How it works :

  • Random generation of big amount of points (voronoi seeds)
  • Computation of Voronoi's diagram with selected seeds
  • Associate the average of the values ​​with the pixels of each region
